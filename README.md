# 🚗 Sarkin Mota Landing Page

A modern landing page prototype built for **Sarkin Mota**, an upcoming motor marketplace focused on making vehicle buying and selling easier, more accessible, and more trustworthy.

This project was created as a concept website to showcase the platform's potential online presence, user experience, and brand direction while the marketplace is still undergoing approval and further development.

---

## About the Project

Sarkin Mota is envisioned as a digital marketplace where individuals and dealerships can connect to buy and sell vehicles with confidence.

At this stage, the project is a **landing page prototype** designed to demonstrate what the future platform could look like. It is not yet connected to a live marketplace, inventory system, or customer database.

The goal of this website is to present the brand, highlight its vision, and provide a glimpse into the experience users can expect when the full platform launches.

---

* Fully responsive design across mobile, tablet, and desktop

## Getting Started

### Clone the Repository

```bash
git clone (https://github.com/KdanTechnies/Sarkin-Mota)
```

### Move into the Project Directory

```bash
cd Sarkin-Mota
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```

---

## Production Build

```bash
npm run build
npm start
```
