# Sarkin Mota Landing Page

A modern, responsive, and visually engaging landing page built for **Sarkin Mota**, a motor marketplace currently under review and approval.

> **Important Notice**
>
> This website is a **demo/testing project** created to showcase the potential online presence and user experience of the Sarkin Mota marketplace. The actual marketplace platform is still undergoing review, approval, and development. Information displayed on this website is for demonstration purposes only.

## Overview

Sarkin Mota aims to become a trusted online marketplace for buying and selling vehicles. This landing page serves as a proof of concept, demonstrating how customers can discover vehicles, learn about the company, and interact with the brand online.

## Features

* Modern and responsive design
* Built with Next.js and TypeScript
* Mobile-first user experience
* Smooth animations using Framer Motion
* Premium automotive-inspired interface
* Featured vehicle showcase
* Brand showcase section
* Customer testimonials
* Call-to-action sections
* SEO-friendly structure

## Tech Stack

* Next.js 15
* TypeScript
* Tailwind CSS
* Framer Motion
* Lucide React
* Responsive Web Design

## Project Status

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate into the project:

```bash
cd sarkin-mota
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```

## Build for Production

```bash
npm run build
npm start
```
